# Finary Patrimoine

MVP web moderne de suivi patrimonial inspiré des usages d'un dashboard wealth-tech, sans reprise de branding ou de contenu propriétaire.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React

## Fonctionnalités MVP

- Landing page simple
- Auth locale mock via localStorage
- Dashboard patrimoine consolidé
- Vue comptes
- Vue actifs
- Allocation patrimoniale
- Transactions récentes
- Objectifs financiers
- Responsive mobile / desktop

## Démarrage

```bash
npm install
npm run dev
```
